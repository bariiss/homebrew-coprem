# homebrew-coprem

Homebrew tap for [coprem](https://github.com/bariiss/coprem) — GitHub Enterprise
Copilot premium-request analytics CLI.

## Install

```sh
brew tap bariiss/coprem
brew trust bariiss/coprem        # one-time: Homebrew 6.0+ requires explicit trust
brew install --cask coprem
```

## Upgrade

```sh
brew upgrade --cask coprem
```

## Uninstall

```sh
brew uninstall --cask coprem
brew untap bariiss/coprem
```
